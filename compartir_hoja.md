# Bot de Gastos - Funcionando

Tu bot está conectado a la hoja "Gastos" y operativo.

## Formato de mensaje:
```
Producto
Lugar  
Categoría
Subcategoría
Importe
Cantidad
```

**Ejemplo:**
```
Pan
Panadería
Comida
Productos básicos
2500
1
```

El bot registra automáticamente la fecha en tu hoja.

Comandos: /start y /help para instrucciones.
